# Imagemagickで画像のDPIを変更する

画像のDPI（dot per inch）がばらばらの場合、
TeXで画像を取り込んだときのサイズがばらばらになるため統一する。

以下はIN.pngをDPI 72にしてOUT.pngに出力するコマンド。

```bash
convert -density 72 -units PixelsPerInch IN.png OUT.png
```

もともとのDPIを調べるコマンド。

```bash
identify -format '%x,%y\n' IN.png
```
